มี “Magic numbers” ใน CSS ไหม? (เช่น margin 37px, font-size 13px แบบแปลก ๆ)
ตอบ มี เช่น max-width: 420px; เป็นต้น

ถ้ามี ให้ปรับเป็นค่าที่มีเหตุผล เช่น 1rem, 1.5rem หรือ 32px เป็นต้น
ตอบ max-width: 26rem; /* อิงจาก font-size */ เป็นต้น
 
มี โค้ดซ้ำ ๆ ไหม? (เช่น style ของ card เหมือนกันหลายที่)
ตอบ มี เช่น style , class , link , href

สร้าง class .card, .btn-primary ฯลฯ แล้วนำไปใช้ซ้ำ
แยกส่วน CSS เป็นหมวด เช่น

/* Layout */, /* Components */, /* Typography */

เขียนสรุปสั้น ๆ 5–10 บรรทัดใน README.md ว่า

คุณเจอ code smell อะไรในงานของตัวเองบ้าง
ตอบ  1.ใช้ <button> แทน <a> เพื่อเปลี่ยนหน้า
    2.ใช้ CSS custom แต่ยังไม่ตั้งชื่อ class ให้สื่อความหมาย
    3.ฟอร์ม Login ใช้ POST แต่ไม่มี backend
    4.ใช้ <h2>, <h4> แบบไม่เป็นลำดับ
    5.ใช้ URL รูปจากภายนอกโดยไม่ควบคุม
    6.ใช้ Bootstrap + CSS แต่ยังไม่มี theme กลาง
คุณแก้ด้วยวิธีใด (อ้างถึงแนวคิดจากในคลาส: SRP, DRY, Magic Number ฯลฯ)
